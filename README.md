## Operators 

| Type    | Operator | Meaning               |
| ------- | -------- | --------------------- |
| Numeric | `-eq`    | equals                |
|         | `-ne`    | not equal             |
|         | `-gt`    | greater than          |
|         | `-lt`    | less than             |
| String  | `=`      | equals                |
|         | `!=`     | not equals            |
| File    | `-f`     | file exists           |
|         | `-d`     | directory exists      |
|         | `-e`     | file/directory exists |
